info How to use Learning Designer

1. Design Structure

Build your course by adding Activities (modules/topics) and filling them with Steps (specific tasks). Reorder items using the drag handles drag_indicator.

2. Learning Types

Select a pedagogical type for each step:

  • Acquisition
  • Collaboration
  • Discussion
  • Investigation
  • Practice
  • Production

3. Analysis Dashboard

Switch to the Analysis tab to visualize the balance of your design. Check the "Pedagogical Balance" radar chart and the "Context" pie charts to ensure your course meets its goals.

4. Save & Export

Save/Load: Keep your work in a .json file.
Export: Generate a structured Markdown file optimized for Generative AI (RAG) or documentation.
